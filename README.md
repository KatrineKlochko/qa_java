# qa_java
QA Java Project
Sprint 6

Этот проект предназначен для автоматизированного тестирования с использованием Java, Maven и Jacoco.

| технология | версия |
|------------|--------|
| Java       | 11     |
| Junit      | 4.13.2 |
| Maven      | 3.8.1  |
| Mockito    | 5.20.0 |
| Jacoco     | 0.8.14 |

Для запуска тестов используйте команду: mvn clean test

Посмотреть отчет о покрытии: mvn jacoco:report

Отчет появится в директории target/site/jacoco/index.html
